Quel type de copyleft voulez vous ?

Copyleft fort : Lorsqu'un projet de logiciel contient une partie de votre code, le projet dans son ensemble doit être distribué sous votre licence, si tant est qu'il soit distribué. Cela aura pour effet que le code source de tous les ajouts faits au code sera disponible.
Copyleft faible : Lorsqu'un projet de logiciel contient une partie de votre code, les parties du projet dont vous êtes à l'origine doivent être distribuées sous votre licence, si elles le sont. D'autres parties peuvent être distribuées sous d'autres licences, même si elles font partie d'un travail qui est - dans son ensemble - une version modifiée de votre code. Cela aura pour effet que le code source de certains ajouts faits au code ne sera pas disponible.
