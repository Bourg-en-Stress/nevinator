Licence ou pas licence ?

Vous ne voulez pas choisir une licence ? Lorsque vous réalisez une œuvre créative (ce qui inclut le code), cette œuvre est soumise par défaut à un droit d'auteur exclusif. À moins que vous n'incluiez une licence spécifiant contraire, personne d'autre ne peut copier, distribuer ou modifier votre travail sans risquer d'être victime d'un vol, d'un détournement ou d'un litige. Interdire l'utilisation de votre code n'est peut-être pas ce que vous entendez par "sans licence". Une licence open source permet la réutilisation de votre code tout en conservant les droits d'auteur. Si votre objectif est de vous affranchir complètement des restrictions de droits d'auteur, essayez plutôt une licence Créative Commons.
